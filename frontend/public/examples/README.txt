Примеры отчётов для онбординга и вкладки «Помощь».
Чтобы картинки отображались, скопируйте в эту папку два PNG-файла с именами:

  cashflow.png — скриншот блока «Денежный поток» (расходы, доходы, разница, график по месяцам)
  capital.png  — скриншот блока «Чистый капитал» и графика «Финансовый путь»

Путь: frontend/public/examples/
После добавления файлов пересоберите фронтенд (npm run build).
